Weather App 

A simple Python command-line weather application that uses the OpenWeatherMap API to display real-time weather information for any city.

---

Features

- Search weather by city name
- Displays:
  - Current temperature
  - Feels-like temperature
  - Weather condition
  - Humidity
- Supports Fahrenheit and Celsius display
- Heat advisory warning for high temperatures

---

Technologies Used

- Python 3
- Requests Library
- OpenWeatherMap API

*IMPORTANT*
You will need to generate your own API key to use in this program

1. Go to the OpenWeatherMap website:
2. https://openweathermap.org/api
3. Create a free account
4. Generate an API key
5. Replace the API key in the code:

API_KEY = "your_api_key_here" <- place your key here(DO NOT SHARE THIS KEY WITH ANYONE FOR SECURITY PURPOSES. THIS IS TO PREVENT THE UNAUTHORIZED ACCESS OF SENSITIVE DATA)
